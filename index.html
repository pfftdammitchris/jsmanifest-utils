set

  • set(obj: any, path: string | number | symbol | Path, value: any, setter?: (obj: any, path: string | number | symbol | Path, value: any) => any): any
    • Defined in utils.ts:640

    Sets the value at path of object. If a portion of path doesn't exist, it's created. Arrays are created for missing index properties while objects are created for all other missing properties.

    If a custom setter function is provided it will be used to set the values

    example
    const object = { 'a': [{ 'b': { 'c': 3 } }] };
    set(object, 'a[0].b.c', 4);
    console.log(object.a[0].b.c); // Result: 4
